### CSS is a Cascading Stylesheet

CSS (Cascading Style Sheets) is a way to add styles to HTML tags. While HTML provides the structure of a webpage through semantic tags, CSS allows you to style those tags visually. 

1. **Cascading**: The term "cascading" refers to how different styles can be applied to the same element. If there are conflicting styles, CSS uses a specific order of precedence to determine which style to apply. Inline styles override internal styles, which in turn override external styles.

2. **Usage**: CSS can be applied in three main ways:
   - **Inline**: Styles are applied directly within the HTML tag using the `style` attribute.
   - **Internal**: Styles are placed within `<style>` tags in the head section of the HTML document.
   - **External**: Styles are written in a separate CSS file and linked to the HTML using the `<link>` tag.

3. **Default Styles**: Browsers apply default styles to HTML elements, but you can overwrite these styles to create a consistent design across your product.

As we progress through this lesson, we'll focus on three critical areas of design: typography, colors, and layout.

Here’s a simple code example for inline, internal, and external CSS:

**Inline CSS:**
```html
<p style="color: blue;">This is a blue paragraph.</p>
```

**Internal CSS:**
```html
<head>
  <style>
    p {
      color: blue;
    }
  </style>
</head>
```

**External CSS:** (in a separate `styles.css` file)
```css
p {
  color: blue;
}
```
